## Photoshop automation tools

## - fileConverter

  *Convert files to different formats using Photoshop*

  **Class usage example:**
        Converter(['png', 'file1.psd', 'file2.psd', 'file3.psd'])

  **Cmd execution example:**
        fileConverter.py "png" "file1.psd" "file2.psd" "file3.psd"
